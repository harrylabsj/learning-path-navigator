# Learning Path Navigator 学习路径导航仪

根据你的学习目标、当前水平和可用时间，生成可执行的个性化学习路径——从目标到成果的全流程导航。

## What it does
- 分析学习目标与现有技能的差距
- 按阶段规划学习序列和时间预算
- 为每个阶段匹配最佳学习资源（免费为主）
- 设置里程碑检查点跟踪进度

## Example scenarios

**1. 零基础转码**
> 👤 我本科学文科，现在想做程序员，网上课太多了不知道从哪开始。
> 🤖 文科转码学习路径：第一步，B 站搜"大学计算机基础"看哈工大/浙大的免费视频，1 个月内学完，同时刷 FreeCodeCamp 的 HTML/CSS/JS 入门；第二步选方向——前端最快（学 React+Next.js，3 个月能找工作）；第三步做项目（模仿做一个 To-Do App 然后自己改需求，GitHub 找开源项目参与）；第四步找工作（刷 LeetCode 热题 100 + 牛客网看面经）。全程不用付费课，B 站+freeCodeCamp+Coursera 免费足够。

**2. 制定 Python 学习计划**
> 👤 帮我制定一个 Python 学习计划，我有编程基础但没系统学过 Python。
> 🤖 8 周 Python 速成路径——第1-2 周：语法与数据结构（B 站黑马程序员视频 + LeetCode 简单题）；第3-4 周：面向对象+文件操作+异常处理（每节学完做一个小练习）；第5-6 周：选一个方向深入（数据分析用 Pandas / 后端用 Flask / 爬虫用 Scrapy）；第7-8 周：完成一个完整的迷你项目并写文档。每周 10 小时预算，每 2 周设一个里程碑检查。

**3. 3 个月学会数据分析**
> 👤 3 个月学会数据分析，规划一下。
> 🤖 12 周数据分析路径——第一阶段（3周）：Python 基础 + NumPy + Pandas 数据处理；第二阶段（4周）：Matplotlib/Seaborn 可视化 + SQL 基础 + 业务指标理解；第三阶段（3周）：完整项目实战（用 Kaggle 数据集完成一个从清洗到可视化到结论的报告）；第四阶段（2周）：简历项目包装 + 模拟面试。建议每周 8-10 小时，项目驱动学习效果最好。
